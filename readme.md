### Uso

### Compilar
Compilar utilizando cmake. Por ejemplo:

```sh
cmake -B build/
cmake --build build/
```

Se encontraran los ejecutables en la carpeta de compilación, build/ en el ejemplo anterior.  
Alternativamente, abrir y compilar con CLion.
